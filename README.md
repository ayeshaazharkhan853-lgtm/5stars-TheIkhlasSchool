The Ikhlas School - Official Website
Welcome to the source code for The Ikhlas School website. 
This project is a multi-page responsive website designed to showcase 
the school campus, academic programs, and its unique approach to
holistic education—blending modern sciences with religious values.

 Key Features
Holistic Curriculum: Dedicated sections for Modern Subjects (Science, Math) 
and Spiritual Learning (Nazra-e-Quran).

Modern UI/UX: A clean, professional look using the Fredoka  font family.
Responsive Layout: Fully compatible with desktop made with CSS .

Interactive Sections:
Success Formula: 
               A unique mathematical approach to student attitude.
Principal's Message:
               A formal leadership introduction with a custom photo frame.
Admission Guide: 
                A step-by-step timeline for new enrollments.
Environment Gallery: 
                A split-layout showcase of the school campus.

 Built With
This project was built using standard HTML  & CSS (only);

├── index.html            # Homepage (Hero Video, Stats, Success Formula)
├── about.html            # About Us (Principal's Message, Vision/Mission)
├── academics.html        # Academics (Modern Subjects + Nazra/Religious Values)
├── admission.html        # Admissions (Schedules, Eligibility, Process)
├── explore.html          # Environment (Campus Gallery)
├── style.css             # Global Styles (Navbar, Footer, Root Colors)
├── academics.css         # Academic page specific styling
├── admission.css         # Admission page specific styling
|── about.css             # About page specific styling
└── images/               # Directory for school photos and icons

Google Fonts: Integrated Fredoka for a friendly feel.
